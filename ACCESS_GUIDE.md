# 企业微信访问指南

## 📱 访问方式总览

### 🌐 推荐方式：浏览器访问

**最佳体验，功能完整**

1. **直接链接**：
   ```
   https://carol-huang00.github.io/seek-stars/
   ```

2. **操作步骤**：
   - 复制链接
   - 在任意浏览器中打开
   - 完全正常使用

3. **特点**：
   - ✅ 所有功能正常
   - ✅ 无安全限制
   - ✅ 页面加载快速
   - ❌ 无法获取企业微信用户信息

### 🏢 企业微信内访问

**适合企业内部推广**

#### 方式一：企业微信内置浏览器

1. **访问步骤**：
   ```
   企业微信聊天 → 文件传输助手 → 粘贴链接 → 点击链接 → "用浏览器打开"
   ```

2. **链接地址**：
   - 标准版：`https://carol-huang00.github.io/seek-stars/`
   - 企业微信版：`https://carol-huang00.github.io/seek-stars/?wecom=true`

3. **特点**：
   - ✅ 便捷访问
   - ⚠️ 可能出现安全提示
   - ❌ 企业微信认证功能受限

#### 方式二：二维码访问

1. **使用提供的二维码生成器**：
   - 打开 `qr-generator.html`
   - 扫描对应的二维码
   - 选择浏览器打开

2. **自定义二维码**：
   ```javascript
   // 在线生成二维码
   const qrUrl = "https://carol-huang00.github.io/seek-stars/";
   ```

### 可信域名问题解决

#### 临时解决方案

1. **跳过域名验证**：
   ```
   在企业微信中打开链接
   → 出现安全提示时
   → 选择"继续访问"
   → 选择"用浏览器打开"
   ```

2. **收藏夹方式**：
   ```
   浏览器中打开 → 添加到收藏夹 → 从收藏夹快速访问
   ```

## 🛠 故障排除

### 常见问题

1. **页面无法加载**：
   - 检查网络连接
   - 尝试切换网络
   - 使用浏览器直接访问

2. **功能异常**：
   - 清除浏览器缓存
   - 尝试其他浏览器
   - 检查浏览器版本

3. **安全提示过多**：

   - 添加信任书签
